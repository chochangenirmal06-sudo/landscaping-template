# GreenEdge Landscaping — Project Overview

## The Big Picture

We are building a **premium, high-converting demo website** for a fictional landscaping business called **GreenEdge Landscaping**. The goal is to nail the quality first — then refactor it into a reusable template for any local business (roofers, electricians, plumbers, detailers, etc.).

---

## Three-Phase Roadmap

| Phase | Goal | Status |
|-------|------|--------|
| **1 — Build the Demo** | Complete, polished GreenEdge site | 🔄 In Progress |
| **2 — Templatize** | One config file → fully re-branded site | ⏳ Later |
| **3 — Write the Playbook** | `.md` docs to replicate for any niche | ⏳ Later |

---

## Phase 1 Scope — GreenEdge Demo

### Business Details (Fictional)
- **Business Name:** GreenEdge Landscaping
- **Tagline:** *"Where Every Edge Is Perfect"*
- **Location:** Austin, Texas
- **Services:** Lawn care, landscape design, irrigation, seasonal cleanup, hardscaping
- **Tone:** Premium, trustworthy, outdoorsy, modern
- **Target Customer:** Homeowners aged 30–60, middle-to-upper income, value quality over price

### Conversion Goals
- Primary CTA: Book a Free Estimate
- Secondary CTA: View Our Work / Call Now
- Trust signals: Reviews, before/after, years in business, licensed & insured

## Working Method

1. **Plan first** — use the `.md` files in this folder as the source of truth
2. **Build section by section** — one section at a time in Claude Code
3. **Iterate until perfect** before moving to the next section
4. **No shortcuts** — every section must feel premium and intentional
